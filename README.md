# 🌐 Subsurface Object Detection Simulator

A physics-based web simulation that models **subsurface object detection using electromagnetic (EM) wave propagation** in layered media.

## 📌 Features

* 📡 EM wave propagation simulation
* 🧱 Layered medium modeling
* 📊 Real-time visualization using Chart.js
* 🎛 Adjustable transceiver parameters
* 🧪 Multiple scenarios:

  * Road Infrastructure
  * Utility Detection
  * Landmine Sweeper
  * Custom Mode

---

## 🧠 Core Concepts

This simulator is based on:

* Electromagnetic wave theory
* Material permittivity & permeability
* Signal reflection & impedance mismatch
* Wave velocity in different media
